# security policy

please use [GitHub's security advisories](https://github.com/tiagozip/cap/security/advisories/new) or email [security@trycap.dev](mailto:security@trycap.dev) ([pgp key](https://github.com/tiagozip.gpg)) to report any security issues.

if you find bots in the wild bypassing Cap Instrumentation's headless browser checks, please email me too so i can take a look.

## ai usage

security advisories found from ai scans are not currently allowed due to large amounts of low-quality issues.

for "CAPTCHA bypasses", a working fix is strictly required.
